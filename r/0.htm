Como bloquear um telefone pelo código IMEI?

Para bloquear o telefone com o código IMEI é 
necessário entrar em contato com a operadora.
 Para realizar essa tarefa, podem ser 
solicitados, além do número do IMEI, 
a confirmação dos seus documentos e,
 às vezes, o número do Boletim de
 Ocorrência registrado na Polícia Civil. 

Os números para fazer essa solicitação com as operadoras são os seguintes:

Operadora	Número
Vivo	        1058
Tim      	1056
Claro	        1052
Oi	        1057
